# Issuely
A basic javascript issue tracker.

<img src="Images/main.png" width="90%"> <img src="Images/search.png" width="90%">

### Usage
Run this on any webserver. Maybe rename index.html though.

Form to submit issue takes in a description, severity and optional team/employee assignment. Every new issue gets assigned a 4 digit ID. 

To search, either type in an id or a description. Multiple are allowed, separated by spaces.
